# ⚽ Sport Portal - פורטל כדורגל לחברים

פורטל לניהול משחקי כדורגל בין חברים - הרשמה למשחקים, חלוקת קבוצות, סטטיסטיקות ועוד.

## התחלה מהירה

### 1. התקנת תלויות
```bash
npm install
```

### 2. הגדרת הסביבה
קובץ `.env` כבר קיים עם הגדרות ברירת מחדל. לייצור, עדכן את:
- `NEXTAUTH_SECRET` - מחרוזת סודית אקראית
- `NEXTAUTH_URL` - ה-URL האמיתי של האתר

### 3. יצירת מסד הנתונים
```bash
npx prisma migrate dev
npx tsx prisma/seed.ts
```

### 4. הרצת השרת
```bash
npm run dev
```

פתח http://localhost:3000

## פרטי כניסה לבדיקה

## תכונות המערכת

### שחקן רגיל
- ✅ הרשמה ופרופיל אישי (עמדה, רמה, טלפון)
- ✅ אישור / ביטול הגעה למשחקים
- ✅ רשימת המתנה אוטומטית כשהמשחק מלא
- ✅ צפייה בחלוקת קבוצות
- ✅ צ'אט/לוח הודעות לפני כל משחק
- ✅ גלריית תמונות למשחקים
- ✅ הצבעה בסקר אחרי המשחק (מי ניצח, MVP, כמה נהנית)
- ✅ לוח סטטיסטיקות עם תארים מצחיקים

### מנהל (Admin)
- ✅ יצירת משחקים עם תאריך, שעה, מיקום, מספר שחקנים
- ✅ סגירת הרשמה
- ✅ חלוקת קבוצות אוטומטית (אלגוריתם איזון לפי עמדה ורמה)
- ✅ פתיחת סקר אחרי המשחק
- ✅ ניהול שחקנים והרשאות

## סטטיסטיקות ותארים
- 🪨 **הסלע** - הנוכחות הגבוהה ביותר
- 💨 **הקישקוש** - הכי הרבה ביטולים
- 🏆 **הנשר** - הכי הרבה ניצחונות
- ⭐ **כוכב העונה** - שחקן המשחק הכי הרבה פעמים

## טכנולוגיות
- **Next.js 16** + TypeScript
- **Tailwind CSS** - עיצוב מודרני RTL
- **Prisma 7** + SQLite (פיתוח) / PostgreSQL (ייצור)
- **NextAuth.js** - אימות משתמשים

## פריסה לאינטרנט (Vercel)
1. העלה את הקוד ל-GitHub
2. חבר ל-Vercel וקשר את הריפו
3. הוסף משתני סביבה ב-Vercel Dashboard
4. עבור למסד נתונים PostgreSQL לייצור (Supabase / Neon)

```env
DATABASE_URL=postgresql://...
NEXTAUTH_SECRET=your-secret
NEXTAUTH_URL=https://your-domain.vercel.app
```
